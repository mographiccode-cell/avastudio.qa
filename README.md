# AVA Studio — Landing Page

صفحة هبوط احترافية لاستوديو AVA Studio في قطر، مصممة بأسلوب سينمائي تفاعلي.

## نظرة عامة
صفحة هبوط إبداعية بصرية مستوحاة من الهوية السينمائية، مصممة خصيصاً لحساب Instagram [@avastudio.qa](https://www.instagram.com/avastudio.qa/).

## المميزات
- تصميم سينمائي بتقنية Scroll-Based Storytelling
- تأثيرات ثلاثية الأبعاد و parallax
- انتقالات كاميرا سينمائية
- دعم كامل للغة العربية RTL
- تجاوب كامل مع جميع الأجهزة
- تكامل مع Instagram Embed API
- أداء عالي وتحسين SEO

## التقنيات المستخدمة
- Tailwind CSS
- Instagram Embed API
- CSS 3D Transforms & Animations
- Vanilla JavaScript للتحكم بالحركة
- GitHub Pages للنشر
