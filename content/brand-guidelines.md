# Brand Guidelines — AVA Studio

## Visual Identity System

### Color Palette
```
Primary Dark     #0a0a0f    — Backgrounds, deep sections
Primary Light    #1a1a2e    — Cards, surfaces
Secondary        #16213e    — Accent backgrounds
Gold Accent      #c9a96e    — CTAs, highlights, icons
Gold Light       #d4af37    — Hover states, gradients
Rose Accent      #e8c4b8    — Warm accents
Text Primary     #f0ece4    — Main text
Text Secondary   #a8a4a0    — Secondary text
Text Muted       #6b6763    — Captions, metadata
Surface Dark     #12121c    — Elevated surfaces
Surface Card     #1a1a28    — Card backgrounds
Border Subtle    #2a2a3a    — Borders, dividers
```

### Typography
- **Headings**: Cairo (Arabic), Playfair Display (English accents)
- **Body**: Tajawal (Arabic), system fonts (English)
- **Direction**: RTL primary, LTR fallback for English segments
- **Font Weights**: 300 Light, 400 Regular, 600 SemiBold, 700 Bold, 900 Black

### Spacing & Layout
- Generous whitespace with cinematic proportions
- Sections separated by atmospheric transitions
- Content width max 1280px with breathing room
- RTL grid with proper padding adjustments

### Motion & Animation
- Scroll-triggered reveals
- Parallax depth layers (3-5 layers)
- Smooth camera-like transitions between sections
- Hover lift effects on cards (translateZ)
- Glow pulse animations on CTAs
- Cinematic fade-ins with blur transitions
- Frame-by-frame reveal for hero sections

### Photography Treatment
- Dark, moody tone with warmth
- Dramatic lighting accents
- Film grain overlay (subtle)
- Light leak effects on section transitions
- Vignette on dark sections

### Iconography
- Minimal line style
- Gold (#c9a96e) color for active/featured
- White (#f0ece4) for standard
- Muted (#6b6763) for inactive
